# Study Buddy

## How to run Study Buddy

```
git clone https://www.github.com/bryankle/group35 study_buddy
```

## How to build:
Navigate into directory study_buddy
1. Install Node
2. Install NPM

To install packages

```
npm install
```

## To start application

```
npm server
```

## Accessing the site

```
http://localhost:3000/
```
